# LorePia 교차 플랫폼 기능·UI QA 체크리스트

- 기준일: 2026-07-24
- 대상: 현재 작업 트리의 `apps/desktop-mobile`
- 플랫폼: iOS Simulator, Android Emulator, macOS
- 상태값: `PENDING`, `PASS`, `FAIL`, `BLOCKED`, `NOT RUN`
- 증거 폴더: `docs/qa/evidence/2026-07-24-cross-platform/`

## 실행 원칙

1. 이 문서를 먼저 고정한 뒤 세 플랫폼에서 같은 ID를 실행한다.
2. 실제 UI 조작 결과와 자동화·단위 테스트 결과를 구분한다.
3. 외부 LLM 호출, 자격증명 저장, 실제 데이터 삭제처럼 테스트 환경에서
   안전하게 실행할 수 없는 항목은 성공으로 추정하지 않고 `BLOCKED` 또는
   `NOT RUN`으로 기록한다.
4. 실패는 재현 절차, 기대 결과, 실제 결과, 플랫폼, 심각도와 함께 보고한다.
5. 테스트 중 변경한 테마·표시 모드는 원래 값으로 복원한다.
